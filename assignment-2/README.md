Multi‑Screen Mobile Application (Expo + React Native)
This project is a multi‑screen mobile application built with Expo, React Native, and TypeScript, recreating the layout and navigation flow of the Uber mobile app. It was developed collaboratively in a group of two as part of the Advanced Mobile Application Development course.
Features
- Bottom Tab Navigation (Home, Browse, Orders, Account)
- Stack Navigation for deeper flows (Account → Personal Info, Security, Privacy & Data, Account Checkup)
- Account Section UI with profile, quick links, and suggestions card
- Dynamic content using reusable components (OrderCard, CategoryIcon, RestaurantInfoNode)
- Multiple functional screens: Home, Browse, Orders, Account, Personal Info, Security, Privacy & Data, Account Checkup, Store, Receipt, Search
Tech Stack
- Expo SDK 54
- React Native + TypeScript
- React Navigation (Bottom Tabs + Native Stack)
- @expo/vector-icons
- react-native-reanimated
- react-native-gesture-handler
- @gorhom/bottom-sheet
- react-native-maps

Running the App
Install dependencies:
npm install

Start the app:
npm start

Open in Expo Go or a simulator.
Collaboration
Both group members contributed through GitHub commits and shared development of screens, navigation, and UI components.
